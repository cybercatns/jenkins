Created with Ansible. All manual actions will be overridden.

Stage 01 – Deploy Infrastructure

  • This stage will create the infrastructure.
  • We use tools like Terraform to automate cloud infrastructure as code.
  • Test infrastructure is identical but scaled down version of the production.
  • Terraform has features like Workspaces, Modules, Variables to maintain TEST, UAT and PROD infrastructure requirements in one code.

Stage 02 – Install / Configure the application

  • Application need be properly installed first. This is done using tools like Ansible.
  • Ansible has features like Playbooks, Roles, Tasks and Template to isolate TEST, UAT and PROD configurations.

Stage 03 – Functional / E2E Testing

  • Functional test suites are created by the developers. Some applications are tested against nonfunctional tests too.
  • End-to-end testing is the final testing before the release.
